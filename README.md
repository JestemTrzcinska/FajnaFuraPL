# FajnaFuraPL

## General info

A student project about renting cars.

## Setup

To run server (localhost: 5000)

```
$ npm install
$ npm run server
```

To run client (localhost: 8080)

```
$ npm install
$ cd client
$ npm start
```

To run both

```
$ npm install
$ npm run dev
```

## Technologies

Project is created with:

- JavaScript
- HTML, CSS
- MongoDB
- Mongoose
- tbc
